Structure

Start a page with the doctype tag and HTML tag

<!DOCTYPE html>
<html>
</html>

The html tag houses all the rest of the document.
There are 2 parts to the document.

  1. The head which has information like the title, how to render the document, and links to scripts and stylesheets. This information is NOT displayed on the webpage.
  2. The body has all the visible information on the page

Add the head and body tags, and you have all you need to start a webpage

<!DOCTYPE html>
<html>
  <head>
  </head>
  <body>
  </body>
</html>

Hello World!

This is a webpage. Webpages documents retrieved by your computer from a web location.

The web location is specified by the URL (e.g. http://www.google.com) that you type into your browser.

The page itself is a text document that is read and rendered by the web browser. The language used for writing this document is called HTML.

HTML

HyperText Markup Language (HTML) is the language used to create the content of web pages. HTML is made up of <tags> which mark how the content inside of them should be handled.

Some examples:

  • <p> - paragraphs
  • <a> - links
  • <ul> - bulleted lists (like this one!)
  • <ol> - numbered lists
  • <h1> - headings
  • <b> - bold text
  • <em> - italic text

Some tags have properties that can be set in the tag. For example, a link's location can be specified with the href attribute.

<a href="http://www.google.com"> is a link that directs to Google.com

Html also lets you create objects on the page like rectangels. For example:

<div class='box' id='box1' > </div>
<div class='box' id='box2' > </div>
<div class='box' id='box3' > </div>
creates the three boxes below.

The DOM

The Document Object Model (DOM) is a convention for representing and interacting with HTML documents, such as this one. Because HTML documents follow a standard convention, a multitude of programming languages can interact with the same document without having to worry about differing implementations of the specific document. This is good.

CSS

Besides just marking content, there are also many ways we can style the page to vary its apperaance.

Cascading Style Sheets (CSS) is the language used to describe how a HTML document should look.

Elements on a webpage can be given classes, and rules can style elements of the same class the same way.

An element that gets its own styling can be given an ids and be styled that way. For example:

<div class='box' id='box1' > </div>

"box" is a class tag and "box1" is an id tag.

Classes are referenced with a "." before the class name. We can specify how we want all "box"s to look through this line of CSS:

.box{ height: 5em; }

ID's are referenced with a "#" before the ID name. We can give each box its own color with the following lines:

#box1{ background-color: black; }
#box2{ background-color: gold; }
#box3{ background-color: silver; }

For a cool example check out CSS Zen Garden. All of these pages have the same HTML with different CSS
